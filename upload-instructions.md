# ADHD 시간관리 프로그램 - GitHub 업로드 가이드

## 필요한 파일 목록

1. **package.json** - 프로젝트 설정
2. **vite.config.js** - Vite 설정
3. **postcss.config.js** - PostCSS 설정
4. **tailwind.config.js** - Tailwind 설정
5. **index.html** - HTML 진입점
6. **src/** - 소스 코드 폴더
   - index.css
   - App.jsx
   - main.jsx
   - components/ 폴더
7. **vercel.json** - Vercel 배포 설정

## Vercel 배포 설정

Framework: Vite
Build Command: npm run build
Output Directory: dist
Install Command: npm install
